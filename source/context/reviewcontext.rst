Chapter Review Questions
========================

1. What is an algorithm?  Give three properties an algorithm must have.

2. In your own words, explain what it means for Python to be *dynamically
   typed*.  How does this differ from a language like C# or Java?

3. What is the Python interactive shell (REPL)?  When is it useful?

4. What is *abstraction* in computer science?  Give one example of an
   abstraction you already use without thinking about it.

5. A friend argues that computer science is really just programming.
   How would you respond?
